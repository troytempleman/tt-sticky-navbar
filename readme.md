# sticky-navbar

Sticky-navbar is a small front-end demo project showcasing a sticky, responsive navbar with dropdown menus and a mobile hamburger toggle.

## Project Structure

sticky-navbar/
├── index.html
├── css/styles.css
├── js/script.js
├── images/logo.svg
└── docs/notes.md

## Features

- Sticky navigation
- Dropdown submenus
- Hamburger menu for mobile
- Simple, reusable structure

## Usage

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Modify `styles.css` or `script.js` to customize the navbar.
