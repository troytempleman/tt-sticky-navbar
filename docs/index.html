Overview

TT Sticky Navbar is a lightweight HTML/CSS/JS demo that showcases a sticky header with dropdown menus, mobile menu controls, and accessible keyboard handling.

Key Features

  • Sticky header with folder-tab style menu items.
  • Dropdown submenus with consistent borders and alignment.
  • Mobile hamburger menu with focus trap and Escape-to-close.
  • Keyboard-only focus styling with :focus-visible fallback.
  • Search input that expands on desktop and integrates into mobile.

Usage

  • Open index.html in the repo root to view the demo.
  • Edit css/styles.css for visual changes.
  • Edit js/script.js for behavior changes.

Project Structure

  • index.html — demo page
  • css/styles.css — styles
  • js/script.js — interactivity
  • docs/index.html — documentation page
